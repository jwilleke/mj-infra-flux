# Secrets Inventory

Audit of all `Secret` resources in the cluster as of 2026-05-09, after the issue #46 migration work landed three previously hand-applied Secrets into the repo.

The categorization tells you, for any Secret you find, **where its source of truth lives** — the most common question when investigating drift, doing a recovery rebuild, or onboarding a new agent.

## Categories

A Secret falls into exactly one of:

| Category | Source of truth | Action on cluster rebuild |
| --- | --- | --- |
| **Auto-generated** | Controller (cert-manager, k3s, k8s SA controller) | Re-issued automatically |
| **Helm-release state** | Helm itself (chart values + release history) | Re-created when the chart is reapplied |
| **In-repo (SOPS)** | `mj-infra-flux@HEAD`, decrypted by Flux's kustomize-controller via `flux-system/sops-age` | Re-applied by Flux on next reconcile |
| **Bootstrap** | Manual one-time apply — must be re-created out-of-band before Flux can do anything | Manual apply (chicken-and-egg) |
| **Hand-applied (drift)** | Operator memory — should be migrated to in-repo SOPS | Re-create or recover from backup |
| **Stale** | Nothing — orphaned, candidate for deletion | n/a |

### Bootstrap (legitimately hand-applied)

These have a chicken-and-egg dependency on Flux itself or on cluster bootstrap. Cannot live in this repo.

| Secret | Why out-of-band |

## Recovery procedure (if `deby` is rebuilt)

In order:

1. **Bootstrap age key**: restore `~/.config/sops/age/keys.txt` from your backup, then `kubectl -n flux-system create secret generic sops-age --from-file=age.agekey=…`.
2. **`flux bootstrap`** the cluster against this repo. Once flux-system Kustomization reconciles, *all* in-repo SOPS Secrets are re-applied automatically.
3. **Re-create hand-applied Secrets** (`authentik/*`, `cloudflare-api-token`) — until they're migrated, this is operator memory or external backup.
4. **Confirm cert-manager re-issues TLS Secrets** by checking `kubectl get certificate -A` shows `Ready: True` for each. ACME rate-limits apply if many certs renew at once.
